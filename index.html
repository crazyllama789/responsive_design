Website and Issues with Website

I chose this website because Don Joses is my favorite Mexican restaurant in Providence, however, their website is confusing and is not attractive to potential consumers and might halter business.

Click here to go to Don Jose Tequila Original Website

Finding Problems:

- The ORDER ONLINE button should be the only thing on the page used to take you to the online order page, but actually, if you click anywhere on the image as well it will take you to the menu

- The Website has a button that says fold “it”.There is no clarification of what ‘it’ is and once you fold it you can’t unfold it

- In the navigation bar under Order Online, it appears the symbol next to it is a drop-down, but no menu drops down and it just takes you to the order online page

- The font on the button is a light grey which makes it hard to read

- The website is not aesthetically pleasing, so users might leave the website thinking Don Joses is a really bad restaurant

- The order online button is not clear it is a button a user might not recall if they order by clicking the button or the order online through the tab on the navigation bar

- There are three order online buttons (the drop-down, the purple one, and the one in the top right), so the user might not know which to click because the purple order online button takes you to the menu which is confusing

Accessibility:

There are a lot of contrast issues with this website. The accessibility website says that the grey font being used at the button has a low contrast between text and background colors which I agree with. It is very hard to read, and the choice to make the font light grey and small does not help the users to get important information off of the website. Things like the phone number or location could be missed. The accessibility website also points out that the phrase “To answer any questions please call us” text is very small. I agree with that and I think users might not even see that the restaurant wants them to call with questions. I think the final comment I want to make on this website is that for the microphone icon they have alternative text, but the alternative text just says ‘icon’. This would be incredibly confusing if someone did need this alternative text to know what this icon actually was.
